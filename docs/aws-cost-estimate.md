# AWS 月額費用見積もり

Quill の現行 AWS リソース構成に基づく月額コスト見積もり。

## リソース構成

| サービス | 構成 |
|---|---|
| Lambda | Python 3.11 / 256 MB / 30s タイムアウト |
| API Gateway | HTTP API（`$default` ルート） |
| S3 | フロントエンド静的ホスティング（パブリックアクセス全ブロック） |
| CloudFront | PriceClass_200 / OAC + API プロキシ |
| CloudWatch Logs | Lambda ログ出力 |
| IAM | Lambda 実行ロール / GitHub Actions OIDC ロール |

## サービス別コスト（低トラフィック: 〜5,000 req/月）

### 1. Lambda

| 項目 | 値 |
|---|---|
| メモリ | 256 MB |
| 想定リクエスト数 | 5,000 回/月 |
| 平均実行時間 | 500 ms |
| 無料枠 | 月 100 万リクエスト + 40 万 GB 秒 |
| **月額** | **$0.00** |

### 2. API Gateway（HTTP API）

| 項目 | 値 |
|---|---|
| 想定リクエスト数 | 5,000 回/月 |
| 料金 | $1.00 / 100 万リクエスト |
| 無料枠 | 月 100 万リクエスト（12 ヶ月） |
| **月額** | **$0.00**（無料枠終了後: 〜$0.005） |

### 3. S3

| 項目 | 値 |
|---|---|
| ストレージ | 〜10 MB（ビルド成果物） |
| GET リクエスト | 〜10,000 回/月 |
| 料金 | $0.023/GB + $0.0004/1000 GET |
| **月額** | **〜$0.004** |

### 4. CloudFront

| 項目 | 値 |
|---|---|
| データ転送 | 〜1 GB/月 |
| リクエスト数 | 〜15,000 回/月 |
| 無料枠 | 月 1 TB 転送 + 1,000 万リクエスト（常時無料） |
| **月額** | **$0.00** |

### 5. CloudWatch Logs

| 項目 | 値 |
|---|---|
| ログ取り込み | 〜50 MB/月 |
| 料金 | $0.50/GB |
| **月額** | **〜$0.025** |

### 6. IAM / OIDC

**$0.00**（無料）

## トラフィック別まとめ

| シナリオ | 月額概算 |
|---|---|
| 低トラフィック（〜5,000 req/月） | **$0.03〜$0.05** |
| 中トラフィック（〜50,000 req/月） | **$0.10〜$0.30** |
| 高トラフィック（〜500,000 req/月） | **$1.00〜$3.00** |

## 備考

- AWS 無料枠（12 ヶ月）が有効な間は、ほぼ全サービスが $0
- CloudFront の常時無料枠（1 TB/月）により CDN コストは実質ゼロ
- DB なし・ステートレス構成のため RDS/DynamoDB 等のコストが発生しない
- コスト増加の主要因は Lambda 実行回数と API Gateway リクエスト数
